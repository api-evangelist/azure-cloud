# Microsoft Azure Cloud (azure-cloud)
A comprehensive collection of Microsoft Azure cloud service APIs covering compute, storage, databases, AI, networking, security, and developer tools. Azure provides IaaS, PaaS, and SaaS delivery models through a global network of datacenters, with REST APIs secured by Microsoft Entra ID authentication.

## Features

| Name | Description |
|------|-------------|
| Global Infrastructure | Operates across 60+ regions worldwide with 99.99% SLA guarantees for most services. |
| Microsoft Entra ID Authentication | All REST APIs use OAuth 2.0 / Microsoft Entra ID for secure, standards-based authentication. |
| Azure Resource Manager | Consistent management layer for deploying and managing all Azure resources via ARM templates and REST. |
| Async Operations | Long-running operations follow the async pattern with polling endpoints for status checks. |
| Throttling and Rate Limits | Per-subscription throttling with remaining request counts returned in response headers. |
| Pagination | List operations return nextLink for cursor-based pagination across large result sets. |
| Multi-Language SDKs | Official SDKs available for .NET, Java, Python, JavaScript/TypeScript, Go, C++, and Rust. |
| Hybrid and Multi-Cloud | Azure Arc extends Azure management to on-premises and multi-cloud environments. |
